# 🌎 Country Guessing Game

## Description

Country Guessing Game is a simple web application built with Streamlit. The player must guess a country based on clues such as its capital city, population, and region.

The application randomly selects a country and displays hints to the user. After entering a guess, the player receives immediate feedback indicating whether the answer is correct or incorrect.

## Features

* Random country selection
* Country clues (capital, population, and region)
* Answer validation
* Instant feedback
* Generate a new country challenge

## Technologies Used

* Python
* Streamlit

## How to Run

1. Install Streamlit:

```bash
pip install streamlit
```

2. Run the application:

```bash
streamlit run app.py
```

3. Open the local URL displayed in the terminal.

## Project Goal

This project was created as a final project for Code in Place. It demonstrates the use of Python data structures, conditionals, user input handling, and web application development with Streamlit.
